# ChatOPC - 一人公司的 AI 伙伴

ChatOPC 是一个专为一人公司（One Person Company）打造的 AI 对话聊天平台，让一个人也能拥有团队级别的能力。

## ✨ 设计特色

本项目的 UI 设计灵感来源于 [Manus.im](https://manus.im/)，采用极简、专业、现代的设计风格：

- **柔和的渐变背景**：温暖的米灰色渐变，营造专业氛围
- **优雅的字体系统**：标题使用衬线体（Crimson Text），正文使用无衬线体（Noto Sans SC）
- **圆角设计语言**：所有组件采用圆角设计，更加友好现代
- **流畅的交互体验**：平滑的动画和过渡效果

## 🚀 快速开始

### 方式一：使用 Python

```bash
# Python 3
python3 -m http.server 3001

# 然后访问 http://localhost:3001
```

### 方式二：使用 Node.js

```bash
# 安装依赖（如果需要）
npm install

# 启动开发服务器
npm run dev
```

## 🎨 核心功能

- 💬 **智能对话输入框**：支持自动调整高度、快捷操作
- 🏷️ **快捷标签**：一键填充常用操作
- 🎯 **功能卡片展示**：清晰展示平台核心功能
- 📱 **响应式设计**：完美适配桌面和移动端
- ⚡ **流畅动画**：优雅的交互动画效果

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- 响应式设计（Mobile First）
- CSS 变量（Design Tokens）
- Intersection Observer API
